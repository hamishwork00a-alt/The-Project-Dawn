# 理性药物设计模拟器 | ABN-QSS × LLM 协同验证案例

## 🎯 案例价值
本案例展示了 **ABN-QSS 自适应平衡网络** 与 **LLM 智能生成** 如何协同工作，实现从「盲盒式发现」到「系统化设计」的药物研发范式革命。

## 🧩 技术架构
- **ABN-QSS 引擎**：模拟生理系统的动态平衡网络
- **LLM 设计师**：基于设计目标生成候选分子（模拟）
- **Streamlit 界面**：提供交互式验证平台
